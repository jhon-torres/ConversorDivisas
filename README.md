# ConversorDivisas
### Descripción: Programa que permite realizar conversiones de monedas, como:
  - **USD** (Dólares Estadounidense)
  - **MXN** (Pesos Mexicanos)
  - **COP** (Pesos Colombianos)
  - **CLP** (Pesos Chilenos)
  - **EUR** (Euro)
